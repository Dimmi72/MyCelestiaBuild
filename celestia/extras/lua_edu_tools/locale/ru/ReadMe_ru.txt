ЧТО ТАКОЕ 'LUA EDU TOOLS' ?
------------------------------------------------------------------------
'Lua Edu TOOLS' это расширение для 'CELESTIA', обеспечивающее:
- графический интерфейс, который упрощает доступ к основным командам
  управления программой и, позволяющей использовать её в полной мере;

- новые возможности, которые недоступны в официальной версии
  (компас, информационный текст/изображение, маркеры расстояний и т.д.).

Графический интерфейс состоит из следующих компонентов:
  > установка даты/времени, регулировка времени (ускорение, замедление);
  > каталог Солнечной системы с классификацией объектов по их типу
    (планеты, спутники, астероиды и т.д.);
  > регулировка рассеянного света;
  > регулировка светимости галактик;
  > кнопки навигации;
  > настройка параметров просмотра;
  > регулировка FOV (угла обзора);
  > отображение пояса астероидов;
  > панель выбора расширений (дополнительных модулей);
  > оверлейная панель с информационным текстом;
  > оверлейная панель с изображением;
  > режим Планетария/Навигации
  > дополнительные кнопки управления направлением перемещения и движения;
  > функция увеличения (предназначена в образовательных целях и отображает
    Солнечную систему в масштабе).
  > маркеры расстояний:
    - квадратный маркер дистанции, расположен в плоскости наблюдателя по
      центру экрана;
    - круглый маркер дистанции, расположен в орбитальной плоскости
      (плоскость эклиптики) относительно выбранного объекта;
  > отображение компаса: (можно изменить его положение на экране кликнув
    по нему мышкой)
    - Долгота/Широта или Азимут/Прямое склонение (клик мышкой для изменения);
    - Переключение режимов Планетарий/Навигация.

  При использовании комбинации клавиш [Shift]+[G] (перейти к поверхности)
  или кнопки "Компас > Планетарий", автоматически включается режим планетария
  и наблюдатель переносится к поверхности выбранного объекта. Продолжительность
  данного перемещения можно задать в конфигурационном файле 'config.lua'
  (см. пункт "КАК НАСТРОИТЬ/ЛОКАЛИЗОВАТЬ 'LUA EDU TOOLS' ?").

КАК УСТАНОВИТЬ 'LUA EDU TOOLS' ?
------------------------------------------------------------------------
1- Распакуйте архив 'lua_edu_tools' в папку 'extras' в подкаталоге
   с установленной Celestia.

2- Файл 'luahookinit.lua' из архива необходимо разместить в корневом
   каталоге 'Celestia'.

3- Откройте в любом текстовом редакторе файл 'celestia.cfg' (он 
   находится в корневом каталоге 'Celestia') и добавьте следующую строку:

		Configuration
		{
		LuaHook "luahookinit.lua" # <-- Добавленная строка
		...

ВНИМАНИЕ : Расширение 'LUA EDU TOOLS 1.2' требует установленной 'Celestia v1.6'.
	   Которую можно найти на следующем ресурсе Интернет:
	   > http://www.shatters.net/celestia/download.html

------------------------------------------------------------------------
КАК ИСПОЛЬЗОВАТЬ 'LUA EDU TOOLS' ?
------------------------------------------------------------------------
- После установки 'LUA EDU TOOLS', при первом запуске 'Celestia',
  графический интерфейс инструментов будет отображен на экране.

- Отображение графической оболочки 'LUA EDU TOOLS' Вы можете
  включать/отключать кликнув по правой стороне панели инструментов или 
  использовать комбинацию клавиш [Shift]+[i] на клавиатуре.

- Стандартное управление программой 'Celestia' (клавиатурные команды,
  управление мышкой) остается в работе, при включенном интерфейсе
  'LUA EDU TOOLS'.

------------------------------------------------------------------------
КАК НАСТРОИТЬ/ЛОКАЛИЗОВАТЬ 'LUA EDU TOOLS' ?
------------------------------------------------------------------------
- Настроить 'LUA EDU TOOLS' очень просто. Его основные параметры описаны
  в файле 'config.lua'. Здесь Вы можете указать языковую локаль, формат 
  времени, раскраску графических элементов, местоположение компаса на
  экране, необходимый набор кнопок управления... и т.п.

- Графический интерфейс уже доступен на нескольких языках.
  Для пользователей Windows/Linux языковая локаль определяется автоматически.
  Для пользователей Mac необходимо редактировать файл 'config.lua'.
  Переведенную версию этого файла (Readme.txt) Вы можете найти в папке
  'locale' в соответствующих папках языка.

- Чтобы добавить информационный текст, необходимо редактировать файл
  'infos/infoText.lua' или создать новый в соответствующей папке языка
  'locale/lang/infoText_lang.lua', где 'lang' соответствует названию языка,
  например 'ru' (русский). Просто добавьте следующую строку для нужного
  объекта:
			Name_of_object = [[ infoText ]];
  Примечание: Если в названии объекта между словами имеются пробелы, то
              его нужно записать таким образом ["Name of object"].

- Чтобы добавить изображение для объекта, необходимо редактировать файл
  'infos/infoImage.lua'. Просто добавьте следующую строку для нужного
  объекта:
	   Name_of_object = { "image_filename_1"; "image_filename_2"; ... }
	где 'image_filename' соответствует пути к файлу. Например,
	    "../images/image_Venus1.jpg"; 
  Примечание: Если в названии изображения для объекта Вы будете
              использовать пробелы между словами, то его нужно записать
              таким образом ["Name of object"].
  Сами изображения разместите в папке 'images'.

- Если Вы желаете внести свой вклад в локализацию 'Lua Edu Tools',
  то создайте ваши файлы:
	"locale/lang/lang.lua";
	"locale/lang/infoText_lang.lua";
	"locale/lang/images/compass_lang.png".
  Сделать это Вы сможете на примере готовых файлов
  ("locale/fr/fr.lua", "locale/fr/infoText_fr.lua", и
   "locale/fr/images/compass_fr.png").
  Файл-заготовка "compass.psd" для файла "compass_fr.png", содержащая
  все необходимые слои "compass.psd", находится в папке 'images'.
  Не забудьте, также, перевести и этот файл Readme.txt.

  Ваши готовые файлы, пожалуйста, отправьте по адресу <vince.gian@free.fr>. Они
  будут включены в официальный релиз 'LUA EDU TOOLS'. Спасибо!
  
  Примечание: Файлы с расширением *.lua Вы можете редактировать простейшим
              текстовым редактором (например, Блокнот Windows). В случае
              применения специфических шрифтов с подчеркиванием или шрифтов
              с символами кириллицы (русский, греческий и т.д.), рекомендуется
              использовать текстовый редактор Notepad2:
              1 - загрузите программу Notepad2: http://www.flos-freeware.ch/notepad2.html
              2 - откройте файл *.lua в Notepad2: File > Open...
              3 - измените кодировку документа на UTF-8: File > Encoding > UTF-8
              4 - отредактируйте файл и сохраните: File > Save

- Также, по желанию, вы можете добавить свои дополнения для пункта "Расширения".
  Для этого создайте файл Lua с названием вашего расширения
  (для примера см. в папке "adds/Political_Borders/Political_Borders.lua")
  и разместите его в созданной папке с таким же названием в каталоге "adds".
  После этого добавьте строку с вашим Lua файлом расширения в файл
  конфигурации "config.lua".
  Теперь при выборе пункта "Расширения" будет отображаться строка с вашим
  дополнением, только все символы подчеркивания '_' будут заменены на
  пробелы ' '.
  
    Lua файл содержит таблицу данных, включая следующие значения:
    
    > objects = { string:object1path, string:object2path, ... }
       - список объектов, отображение которых можно будет вкл./откл.
       
    >	labelcolor = {number:red, number:green, number:blue, number:alpha}
       - определение цвета для названий дополнения. Названия дополнений
       будут отображаться таким же цветом в окошке "Расширения".
         
    >	locationtype = string:location_type
       - определение типа локации, которую можно будет вкл./откл.
       совместно с отображаемым объектом дополнения.
       Со список возможных типов локации Вы можете ознакомиться по адресу:
       http://en.wikibooks.org/wiki/Celestia/SSC_File#Type_.22string.22
